Variables

Definition

Variables are containers for storing data values. In JavaScript, you can declare variables to hold values that can be used and manipulated throughout your code.

Variable Types

var - The original way to declare variables (function-scoped)
let - Modern variable declaration (block-scoped)
const - For values that shouldn't change (block-scoped)

Syntax

// Variable declarations
var name = "John";
let age = 30;
const PI = 3.14;

// Variable reassignment
name = "Jane";
age = 31;
// PI = 3.15; // This would cause an error (const can't be reassigned)

Practice Question

What will be the output of the following code?

let x = 10;
{
  let x = 20;
  console.log(x);
}
console.log(x);
The output will be:
20 (from inside the block)
10 (from outside the block)
This demonstrates block scoping with let.

Operators

Definition

Operators are symbols that perform operations on variables and values. JavaScript includes various types of operators for arithmetic, comparison, logical operations, and more.

Operator Types

Arithmetic Operators - For mathematical operations (+, -, *, /, %, **)
Assignment Operators - For assigning values (=, +=, -=, *=, etc.)
Comparison Operators - For comparing values (==, ===, !=, !==, >, <, >=, <=)
Logical Operators - For logical operations (&&, ||, !)
Ternary Operator - A shorthand for if-else (condition ? expr1 : expr2)

Syntax

// Arithmetic operators
let sum = 10 + 5; // 15
let remainder = 10 % 3; // 1

// Comparison operators
let isEqual = (10 == "10"); // true (value equality)
let isStrictEqual = (10 === "10"); // false (value and type equality)

// Logical operators
let andResult = (true && false); // false
let orResult = (true || false); // true

// Ternary operator
let age = 20;
let status = (age >= 18) ? "Adult" : "Minor";

Practice Question

What will be the value of 'result' after this code executes?

let x = 5;
let y = 10;
let result = (x > 3 && y < 15) ? "Valid" : "Invalid";
The value of 'result' will be "Valid" because both conditions (x > 3 and y < 15) are true.

Functions

Definition

Functions are reusable blocks of code that perform a specific task. They help organize code into logical, reusable parts and can accept parameters and return values.

Function Types

Function Declarations - Traditional function definition (hoisted)
Function Expressions - Functions assigned to variables
Arrow Functions - Compact ES6 syntax (lexical 'this')
IIFE (Immediately Invoked Function Expressions) - Functions that run immediately

Syntax

// Function declaration
function greet(name) {
  return `Hello, ${name}!`;
}

// Function expression
const greet = function(name) {
  return `Hello, ${name}!`;
};

// Arrow function
const greet = (name) => `Hello, ${name}!`;

// IIFE
(function() {
  console.log("This runs immediately");
})();

// Function with default parameters
function multiply(a, b = 1) {
  return a * b;
}

Practice Question

Convert the following function declaration to an arrow function:

function calculateArea(width, height) {
  return width * height;
}
Arrow function version:
const calculateArea = (width, height) => width * height;

Events

Definition

Events are actions or occurrences that happen in the system you're programming, which the system tells you about so you can respond to them in some way if desired.

Common Event Types

Mouse Events - click, dblclick, mouseover, mouseout, mousedown, mouseup
Keyboard Events - keydown, keyup, keypress
Form Events - submit, change, focus, blur
Window Events - load, resize, scroll

Syntax

// Adding an event listener (recommended approach)
document.getElementById("myButton").addEventListener("click", function() {
  alert("Button clicked!");
});

// Inline event handler (not recommended)
<button onclick="alert('Clicked!')">Click me</button>

// Event object
document.addEventListener("click", function(event) {
  console.log(`Clicked at (${event.clientX}, ${event.clientY})`);
});

// Removing an event listener
function handleClick() {
  console.log("Button clicked");
}
const button = document.getElementById("myButton");
button.addEventListener("click", handleClick);
// Later...
button.removeEventListener("click", handleClick);

Practice Question

Write JavaScript code that changes the background color of a div with id "colorBox" to red when it's clicked.

Solution:
document.getElementById("colorBox").addEventListener("click", function() {
  this.style.backgroundColor = "red";
});

Objects

Definition

Objects are collections of key-value pairs where each key (property) has a value that can be any data type, including other objects or functions (methods).

Object Features

Properties - Key-value pairs that store data
Methods - Functions that are properties of an object
Constructors - Functions that create new objects
Prototypes - Mechanism for inheritance in JavaScript

Syntax

// Object literal syntax
function Person(name, age) {
  this.name = name;
  this.age = age;
  this.greet = function() {
    console.log(`Hello, I'm ${this.name}`);
  };
}
const alice = new Person("Alice", 30);

// ES6 Classes
class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

  greet() {
    console.log(`Hello, I'm ${this.name}`);
  }
}

// Object methods
const person = {
  name: "Bob",
  age: 25,
  greet() {
    console.log(`Hi, I'm ${this.name}`);
  }
};

// Object property access
console.log(person.name); // Dot notation
console.log(person["name"]); // Bracket notation

Practice Question

Create an object called 'car' with properties: make (string), model (string), year (number), and a method startEngine() that logs "Vroom!" to the console.

Solution:
const car = {
  make: "Toyota",
  model: "Camry",
  year: 2020,
  startEngine() {
    console.log("Vroom!");
  }
};

// Usage:
car.startEngine(); // Logs "Vroom!" to console

Arrays

Definition

Arrays are ordered collections of values that can hold multiple items under a single variable name. Each value has a numeric index starting from 0.

Array Methods

Modification Methods - push(), pop(), shift(), unshift(), splice()
Iteration Methods - forEach(), map(), filter(), reduce(), find()
Search Methods - indexOf(), includes(), findIndex()
Transformation Methods - join(), slice(), concat(), sort(), reverse()

Syntax

// Array creation
const fruits = ["Apple", "Banana", "Orange"];
const numbers = new Array(1, 2, 3);

// Accessing elements
console.log(fruits[0]); // "Apple"
console.log(fruits.length); // 3

// Modifying arrays
fruits.push("Mango"); // Adds to end
fruits.pop(); // Removes from end
fruits.unshift("Strawberry"); // Adds to start
fruits.shift(); // Removes from start

// Iteration
fruits.forEach(fruit => console.log(fruit));
const upperFruits = fruits.map(fruit => fruit.toUpperCase());

// Searching
const index = fruits.indexOf("Banana"); // 1
const hasApple = fruits.includes("Apple"); // true

// ES6 Spread operator
const moreFruits = [...fruits, "Grape", "Kiwi"];

Practice Question

Given an array of numbers [1, 2, 3, 4, 5], write code to create a new array with each number squared.

Solution using map():
const numbers = [1, 2, 3, 4, 5];
const squaredNumbers = numbers.map(num => num * num);
console.log(squaredNumbers); // [1, 4, 9, 16, 25]
Alternative solution using forEach():
const numbers = [1, 2, 3, 4, 5];
const squaredNumbers = [];
numbers.forEach(num => {
  squaredNumbers.push(num * num);
});
console.log(squaredNumbers); // [1, 4, 9, 16, 25]
